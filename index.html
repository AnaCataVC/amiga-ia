AmigaIA

A centralized repository for storing skills, agent definitions, and lifecycle hooks. 100% cross-compatible with Antigravity (Gemini) and Claude Code.

Explore Arsenal GitHub Repo
Antigravity
Claude Code

Modular Skills

Shared capabilities stored in skills/ using YAML frontmatter and Markdown. Acts as a dynamic engine for Antigravity.

Dynamic Subagents

Store agent personas in agents/. Claude loads them statically, while Antigravity uses them dynamically to spawn specialized swarms on the fly.

Persistent Memory

Long-term project knowledge and Architectural Decision Records (ADRs) are stored in docs/, giving agents continuous context.

Unified Settings

settings.json manages cross-platform hooks for Claude, seamlessly co-existing with Antigravity's atomic Planning Mode pipeline.

Quick Start

1

Fork this repository to your GitHub account.

2

Clone it to your local machine (e.g., inside your Repos folder):

git clone https://github.com/YOUR_USER/amiga-ia.git
3

Link the folders to your global Antigravity / Claude config as described in the README.
