AmigaIA

or ami for friends

A centralized repository for declarative JSON skills, agent profiles, and their Universal Adapter. 100% cross-compatible with Antigravity (Gemini) and Claude Code.

Explore Tools Quick Start GitHub Repo NPM Package

Modular Skills

Shared capabilities stored in skills/ using Markdown with YAML frontmatter. Acts as a portable action engine for any AI.

Dynamic Subagents

Declarative agent profiles stored as Markdown in agents/. Free of proprietary tools, they use natural language descriptions mapped by the adapter.

Persistent Memory

Long-term project knowledge and Architectural Decision Records (ADRs) are stored in docs/, giving agents continuous context.

NPM & CLI Wizard

Distributed as an NPM package with a built-in amiga-ia-setup CLI wizard for easy, one-click environment configuration.

XML Lazy Loading

Dynamic scanning that generates an XML catalog, loading full Markdown files only when requested by the AI.

Safe Namespacing

All tools strictly enforce the ami- prefix to prevent namespace collisions with external AI ecosystems.

Quick Start

Choose your preferred installation method:

Method A: Native Plugin

Antigravity: Run in terminal: agy plugin install https://github.com/AnaCataVC/amiga-ia

Claude Code: Run inside a session:
/plugin marketplace add AnaCataVC/amiga-ia /plugin install amiga-ia@amiga-ia

Method B: NPM Package

1

Ensure you have Node.js installed on your machine.

2

Install the package globally via NPM: npm install -g @anacatavc/amiga-ia

3

Run the interactive wizard by typing amiga-ia-setup in your terminal to automatically copy the files to your AI configuration folders.
